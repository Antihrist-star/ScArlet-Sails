# Scarlet Sails Website

Interactive web interface for the Scarlet Sails Algorithmic Trading System.

## Files

- **index.html** - Landing page with system overview
- **models.html** - Detailed model descriptions (P_rb, P_ml, P_hyb)
- **dashboard.html** - Real-time metrics and performance dashboard
- **api.html** - REST API documentation
- **styles.css** - Modern styling with dark theme
- **script.js** - Interactive features and animations

## Features

✅ Responsive design
✅ Dark theme
✅ Smooth animations
✅ Model comparison
✅ Performance metrics
✅ API documentation

## Local Development

```bash
python -m http.server 8000
# Open http://localhost:8000
```
